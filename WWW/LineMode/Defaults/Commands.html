Commands

At the prompt level, the following commands are available at the prompt within www . Some are disabled when not applicable. All commands may be abreviated to a single letter. Case is not significant. A number by itself selects a reference from within the text. The document referred to is displayed.
List
The List command gives a numbered list of the references encountered so far, giving the document title (or address if no title). To select one, type the number by itself as above.
Back
Goes back to the document you were reading before.
Home
Go back to the first document you were reading.
Recall
Without a parameter, gives a numbered list of the documents you have visited. To select one, give the recall command with its number as a parameter.
Top
Returns to the top of the present document being viewed.
Help
Gives you basically this list.
Quit
Leaves the application.
