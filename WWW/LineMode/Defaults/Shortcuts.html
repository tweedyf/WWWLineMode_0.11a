Shortcuts

To save having always to go through the www default page, you can use a shortcut to get to an index. On cernvax, the following shell commands are predefined:
yellow
The CERN yellow pages telephone directory.
xfind
The CERN computer center documentation keyword index
vmnews
The CERN computer centre news service: all news articles, in reverse chronological order.
You can search an index which has a shortcut command by giving the keywords as parameters, eg yellow taxes To make your own shortcuts, you can define your own commands. On unix (with csh), use an alias (see VMS equiavlent): alias z www address where address is the hypertext address of the document you want to jump into. You can find the hypertext address of a node from the LIST command or the RECALL command when you are in www. (For nodes with titles, the RECALL command shows you that instead).

An alternative method uses a shell script.
